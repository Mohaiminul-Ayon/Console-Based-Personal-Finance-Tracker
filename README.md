# 💰 Console-Based Personal Finance Tracker

A simple **Console-Based Personal Finance Tracker** built with **Java**.  
This application allows users to manage their personal income and expenses using basic CRUD (Create, Read, Update, Delete) operations.

---

## 🚀 Features

- ➕ Add a new transaction
- 📋 View all transactions
- ✏️ Update an existing transaction
- ❌ Delete a transaction
- 💵 Calculate current balance
- 📅 Use today's date or enter a custom date
- 🆔 Automatically generates unique transaction IDs

---

## 🛠️ Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Encapsulation
- Scanner
- LocalDate

## 📌 Transaction Information

Each transaction contains:

- ID
- Type (Income / Expense)
- Amount
- Description
- Date

---

## 📊 Balance Calculation

The application calculates the current balance using the following formula:

```
Balance = Total Income - Total Expense
```
